快捷键

本页面包含了 DyNode 中所有的常用快捷键与操作方式。

谱面编辑 & 预览

快捷键作用
W增加音乐播放速度
S减小音乐播放速度
A / 滑轮上滚谱面预览后退
D / 滑轮下滚谱面预览前进
Q / Ctrl+滑轮下滚减小音符下落速度
E / Ctrl+滑轮上滚增加音符下落速度
Ctrl+滑轮上滚/滑轮下滚批量调整音符宽度（选择音符时）
Ctrl微调音符宽度（放置音符时）/ 多选音符
Ctrl+A全选音符
C / V调整节拍细分
G调整节拍细分分组
J调整拍线样式
1~5调整编辑模式：Note / Chain / Hold / Edit / Playback
Ctrl+F切换侧面类型：MULTI / MIXER
Z切换纵向（时间）网格吸附
X切换横向（位置）网格吸附
↑ 或鼠标右键拖拽切换编辑侧：DOWN / LEFT / RIGHT
↓ / ← / →调整三面节拍细分线不透明度：0% / 50% / 100%
F透明化非编辑侧音符
H切换三面时间对齐高亮线显示
Ctrl+1/2批量设置音符（不包含 Hold）类型为：Note / Chain
Ctrl+C复制宽度（选择单个音符） / 复制音符
Ctrl+D快速复制音符到下一个节拍线上
Ctrl+V进入粘贴模式（不选择音符） / 批量设置宽度（选择音符）
Ctrl + ←/→ 或 Alt + 滑轮上滚/滑轮下滚切换粘贴/吸附中心音符
Ctrl(+Alt)+方向键微调选中音符位置
M镜像选中/粘贴音符
Ctrl+M镜像复制选中音符
R侧面对称选中音符
Ctrl+R侧面对称复制选中音符
Esc退出粘贴模式
Delete / Backspace删除音符
Delete + Backspace删除全部音符
Enter回放谱面
Ctrl+Z撤销操作
Ctrl+Y还原操作
Ctrl+O/P调整谱面难度
Ctrl+T设置谱面标题
N切换简化模式
Tab切换工具栏显示
右Alt显示全部音符信息

B切换异侧多选属性绑定

延迟调整

关于延迟调整功能更加详细的说明，见 谱面编辑/延迟调整 。

快捷键作用

Timing

快捷键作用
T复制最后一个 Timing Point 到选定音符的时间处
Y添加自定义的 Timing Point
Ctrl+Delete删除选中 Note 所处时间的 Timing Point

界面

快捷键作用

P打开/关闭编辑模式下默认隐藏分数板
O打开/关闭粒子效果
Ctrl+B切换 Bar/Time 显示

常用命令

使用 . 快速输入命令。使用 / 打开控制台。

使用 ESC 或 .q 关闭控制台。

关于命令的详细用法见 高级操作/命令。

命令缩写参数描述
widthwid, w<实数>修改选中音符的宽度
positionpos, p<实数>修改选中音符的位置
sides<0|1|2>修改选中音符的侧面
snap[pre|post|nearest]吸附目标音符到节拍线上
duplicatedup[实数]以指定次数快速复制音符
deduplicatededup/对目标音符进行去重操作
expre<字符串>对选中音符执行表达式
linearlin可变曲线生成 - 线性
cosinecos可变曲线生成 - 余弦
cubiccub可变曲线生成 - 自然三次样条
catromcrom可变曲线生成 - Catmull-Rom 样条
centercen/将选中音符的横向范围中线居中对齐
randomizerand/随机化谱面
purge/清空谱面音符
fix/对谱面音符进行常见修正
