    Shortcuts

    This page contains all the common shortcuts and operations in DyNode.

    Chart Editing & Preview

    ShortcutFunction
    WIncrease music playback speed
    SDecrease music playback speed
    A / Mouse Wheel UpRewind chart preview
    D / Mouse Wheel DownFast-forward chart preview
    Q / Ctrl+Mouse Wheel DownDecrease note falling speed
    E / Ctrl+Mouse Wheel UpIncrease note falling speed
    Ctrl+Mouse Wheel Up/DownBatch adjust note width (when notes are selected)
    CtrlFine-tune note width (when placing notes) / Multi-select notes
    Ctrl+ASelect all notes
    C / VAdjust beat subdivision
    GAdjust beat grouping
    JChange beat line style
    1 ~ 5Switch editing mode: Note / Chain / Hold / Edit / Playback
    Ctrl+FToggle side type: MULTI / MIXER
    ZToggle vertical (time) grid snap
    XToggle horizontal (position) grid snap
    ↑ (or Right-click Drag)Switch editing side: DOWN / LEFT / RIGHT
    ↓ / ← / →Adjust beat line opacity for the three sides: 0% / 50% / 100%
    FFade out non-editing side notes
    HToggle display of time-aligned highlight lines on all sides
    Ctrl+1/2Batch set note type (excluding Hold) to: Note / Chain
    Ctrl+CCopy width (when a single note is selected) / Copy note
    Ctrl+DQuickly duplicate notes to the next beat
    Ctrl+VEnter paste mode (when no note is selected) / Batch set width (when notes are selected)
    Ctrl + ←/→ or Alt + Mouse Wheel Up/DownToggle the snap center note
    Ctrl(+Alt)+Arrow KeysFine-tune the position of selected notes
    MMirror selected/pasted notes
    Ctrl+MMirror-copy selected notes
    RSymmetrically align selected notes across sides
    Ctrl+RSymmetrically duplicate selected notes
    EscExit paste mode
    Delete / BackspaceDelete note
    Delete + BackspaceDelete all notes
    EnterPlay chart
    Ctrl+ZUndo
    Ctrl+YRedo
    Ctrl+O/PAdjust chart difficulty
    Ctrl+TSet chart title
    NToggle simplified mode
    TabToggle toolbar display
    Right AltShow full note information

    Advanced Editing

    ShortcutFunction
    LToggle default width mode
    BToggle binding of multi-select properties across sides
    KSet default width
    `Set custom beat subdivision
    0Execute expression

    Delay Adjustment

    For more details on delay adjustment, see Chart Editing/Delay Adjustment.

    ShortcutFunction
    -/= U(Quickly) adjust the time of all notes in the chart
    Ctrl + -/=Adjust the music playback delay of the current device

    Timing

    ShortcutFunction
    TCopy the last Timing Point to the time of the selected note
    YAdd a custom Timing Point
    Ctrl+DeleteDelete the Timing Point at the time of the selected note

    Interface

    ShortcutFunction
    Middle MouseShow chart statistics
    Ctrl+HToggle hit sounds in editing mode
    PToggle the score board in editing mode
    OToggle particle effects
    Ctrl+BToggle Bar/Time display

    Function Keys & Import/Export

    ShortcutFunction
    Ctrl+NNew project
    Ctrl+SSave project
    Ctrl+Shift+SSave project as
    F1Open project
    F2Import chart file
    F3Import music file
    F4Import background file
    Ctrl+F4Reset all backgrounds
    F5Export chart as XML for devices
    F6Export XML chart for Dynamaker-modified
    Ctrl+F6Randomize chart (auto-save will be disabled)
    F7Fullscreen
    F8Toggle auto-save functionality
    F9Switch theme
    F11Open debug info
    F12Open shortcut list
    Ctrl+F12Screenshot
